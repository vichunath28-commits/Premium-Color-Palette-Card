AweddenStudio Colour Details Artwork Generator (Illustrator Script)

The AweddenStudio Colour Details Artwork Generator is a professionally developed Adobe Illustrator ExtendScript (JSX) designed to automate the process of creating structured and presentation-ready color palette cards.

This script converts raw color inputs into a clean, aligned, and fully editable artwork layout within seconds. It is built for graphic designers, brand designers, UI designers, and presentation creators who need consistent and professional color documentation.

What the script does

Generates a 1080 x 1350 pixel artwork
Creates color palette cards in horizontal and vertical layouts
Automatically displays colour name, HEX code, RGB values, and CMYK values
Aligns all text elements properly within each color block
Maintains professional spacing, hierarchy, and typography
Produces fully editable output inside Illustrator

Smart colour input system

The script supports three workflows

Selected Object Fill
Reads fill colors from selected objects in the active document
Swatch
Uses Illustrator swatches for accurate color selection
Manual RGB
Allows manual entry of RGB values

Colour naming system

Uses an embedded color dataset
Matches RGB values to the closest available color name
Works completely inside Illustrator without external dependencies
Can be extended by modifying the dataset

Technical foundation

This script is built strictly within Adobe Illustrator ExtendScript capabilities. It creates artboards, draws shapes, applies layout logic, adds text, and uses ScriptUI for interaction. It does not rely on external tools or unsupported automation.

Why this script is useful

Designers often spend significant time creating color palette cards manually. This script reduces that effort to a few seconds while ensuring clean alignment and consistent output suitable for professional use.
